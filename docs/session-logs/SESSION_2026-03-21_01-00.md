### SLE-43 — validate_emd_bma.py [DONE — Gate NOT Cleared]

**Result:** EMD+BMA is worse than baseline across all 6 configurations tested.

| Config | BSS | Delta |
|--------|-----|-------|
| tp=0.5, BMA (original) | −0.05899 | −0.06002 |
| tp=0.0, BMA | −0.03518 | −0.03621 |
| tp=0.5/1.0/2.0, no-bma | −0.02741 | −0.02844 |
| tp=0.0, no-bma | −0.02523 | −0.02626 |

**Root cause (3 failures):**
1. EMD degenerates to L1 — both query and candidate share identical RETURN_WINDOWS
   time coordinates, so `time_penalty` is structurally inert. Confirmed by identical
   BSS across tp=0.5/1.0/2.0.
2. Euclidean pre-filter introduces selection bias — top-50 from EMD re-ranking of
   Euclidean top-500 differs from production's pure Euclidean top-50 from all 175k rows.
3. BMA overfits — 50 mixture components on 2000 binary cal samples is overparameterised.

**Action:** Gate not cleared. Phase C EMD work (SLE-47) deferred. Research package ABCs
remain valuable as pluggable architecture. EMD redesign path documented in
`research/phase_c_roadmap.md`.

### SLE-44 — Feature set comparison [DONE — Gate NOT Cleared]

**Result:** returns_only(8) confirmed optimal. Neither alternative clears the gate.

| Feature Set | Mean BSS | 2024 BSS | AvgK (2024) |
|-------------|----------|----------|-------------|
| returns_only (locked) | −0.00770 | +0.00103 | 41.9 |
| returns_overnight | −0.00739 | −0.00023 | 32.2 |
| returns_session | −0.00821 | −0.00075 | 30.1 |

**Root cause:** Curse of dimensionality. `max_distance=1.1019` was calibrated for 8D
space. Adding overnight/session features grows distances, shrinking effective neighborhoods.
2022 Bear AvgK drops to 8.5 (overnight) and 7.2 (session) — statistically unreliable.

**Action:** Locked settings confirmed with provenance. Future: AvgK-adaptive thresholding
could enable higher-dimensional feature sets (Phase C).

**Runtime note:** Each feature set took 45–60 min on full 6-fold walk-forward.
